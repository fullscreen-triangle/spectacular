Quick Example

from spectacular import DiadochiOrchestrator, DomainExpertise, IntegrationPattern

# Initialize the orchestrator
diadochi = DiadochiOrchestrator()

# Define domain expertise
viz_expertise = DomainExpertise(
    domain="data_visualization",
    description="Expert in D3.js and visualization principles",
    keywords=["chart", "graph", "d3", "svg", "interactive"],
    reasoning_patterns=["visual encoding", "chart selection"]
)

# Register domain expert
diadochi.register_domain_expert("visualization", model, viz_expertise)

# Create integration pattern
diadochi.create_integration_pattern(
    IntegrationPattern.MIXTURE_OF_EXPERTS, 
    "moe_pattern"
)

# Process multi-domain query
result = await diadochi.process_query(
    "Create an accessible scatter plot with statistical significance indicators"
)

Quick Start Guide

1

Installation

Clone the repository and install dependencies

git clone https://github.com/your-username/spectacular.git
cd spectacular
make setup
pip install -r requirements.txt
2

Configuration

Set up your environment and API keys

cp .env.example .env
# Edit .env with your configuration
python scripts/init_mzekezeke_network.py
python scripts/init_zengeza_mdp.py
python scripts/init_pretoria_rules.py
3

Run Demo

Test the system with built-in demonstrations

# Run main visualization demo
python -m spectacular.main

# Test Diadochi model combination
python -m spectacular.main diadochi

# Interactive mode
python -m spectacular.main interactive
4

Basic Usage

Start generating visualizations from natural language

from spectacular import SpectacularSystem

# Initialize the system
system = SpectacularSystem()
await system.initialize()

# Generate visualization
result = await system.generate_visualization(
    "Create an interactive scatter plot showing correlation between temperature and sales"
)

print(result['generated_code'])  # D3.js code
print(result['overall_confidence'])  # Confidence score
